📦 mctdocx  
MinecraftConnectTool Docx 文档库，基于 VitePress，支持实时更新。

🎯 特性  
- 📚 清晰的文档结构  
- 🎨 可定制主题与样式  
- 📱 响应式设计  
- 🔍 全文搜索  
- 🚀 快速构建与部署  

📁 目录结构  

```
.
├── .vitepress/          # VitePress 配置
│   ├── config.mts
│   └── theme/
│       └── style.css
├── pages/               # 文档页面
│   ├── index.md
│   ├── quick-start/
│   ├── advanced/
│   ├── deployment/
│   └── examples/
├── dist/                # 构建结果
├── package.json
├── edgeone.json
└── .gitignore
```

🛠️ 开始使用  
1. 克隆仓库  
   
```bash
   git clone https://github.com/linfon18/mctdocx.git
   cd mctdocx
   npm install
   ```

2. 本地开发  
   
```bash
   npm run dev
   ```

3. 构建  
   
```bash
   npm run build
   ```

4. 预览构建结果  
   
```bash
   npm run preview
   ```

📖 文档结构  
- quick-start/：快速开始  
- advanced/：高级用法  
- deployment/：部署指南  
- examples/：示例
